# School-Based Marks Management System

## Overview
This system is designed for secondary schools to manage internal school-based assessments such as:
- Tests
- CATs (Continuous Assessment Tests)
- Quizzes
- School-based evaluations

It is NOT a replacement for CAMIS.
It is intended to help schools record, calculate, and print internal reports before final submission to CAMIS.

## Users
This system is strictly for internal school use:
- Head Teacher
- Director of Studies (DOS / Academic Master)
- Teachers

There is NO access for:
- Students
- Parents

## Main Features
- Teacher login
- Class and subject management
- Test and CAT marks entry
- Automatic calculation of averages
- Internal report card generation (PDF)
- Secure, role-based access

## Technology (100% Free Tools)
- Backend: Node.js + Express
- Frontend: React
- Database: PostgreSQL (Supabase)
- Hosting: Free cloud services
